# 客道 KeDao - 颜色方案

## 设计理念

儒商雅致，墨绿鎏金 - 融合传统文化与现代商务的配色体系

## 主色系统

### 日间模式 (Light Mode)

| 颜色名称 | 色值 | 用途 | Tailwind Class |
|---------|------|------|----------------|
| **墨绿** (Ink) | `#004D40` | 主色调，用于按钮、链接、品牌色 | `bg-primary` `text-primary` |
| **亮墨绿** (Ink Light) | `#00695C` | 辅助强调色，hover 状态 | `bg-accent` `text-accent` |
| **鎏金** (Gold) | `#D4AF37` | 次要强调色，等级标识、重要提示 | `bg-secondary` `text-secondary` |
| **浅金** (Gold Light) | `#F9F1D8` | 轻量背景，信息提示框 | `bg-muted` |
| **宣纸白** (Rice) | `#F5F7FA` | 页面背景 | `bg-background` |
| **警示红** (Danger) | `#C62828` | 错误、警告、紧急状态 | `bg-destructive` |

### 夜间模式 (Dark Mode)

| 颜色名称 | 色值 | 用途 | Tailwind Class |
|---------|------|------|----------------|
| **静夜蓝** (Night) | `#0F172A` | 深色背景 | `bg-background` (dark) |
| **鎏金** (Gold) | `#D4AF37` | 主强调色 | `bg-primary` (dark) |
| **亮墨绿** (Ink Light) | `#00695C` | 次要强调色 | `bg-secondary` (dark) |
| **玻璃灰** | `rgba(15, 23, 42, 0.85)` | 卡片、弹窗背景 | `bg-card` (dark) |

## 字体系统

```css
--font-serif: "Noto Serif SC", "Playfair Display", serif;  /* 宋体 - 标题 */
--font-sans: "Noto Sans SC", "Inter", sans-serif;          /* 黑体 - 正文 */
```

### 使用建议

- **标题 (h1-h6)**: 自动使用宋体 (Noto Serif SC)
- **正文**: 默认使用黑体 (Noto Sans SC)
- **英文数字**: 使用 Playfair Display (衬线) 或 Inter (无衬线)

## 特殊效果

### 玻璃拟态 (Glassmorphism)

```html
<!-- 日间玻璃 -->
<div class="glass">...</div>

<!-- 夜间玻璃 -->
<div class="glass-dark">...</div>
```

### 呼吸动画 (夜间鎏金光效)

```html
<div class="breathing-glow">...</div>
```

### 阴影系统

```css
--shadow-soft: 轻量阴影，用于卡片
--shadow-elevated: 中等阴影，用于悬浮元素
--shadow-float: 强阴影，用于焦点卡片
--shadow-gold: 鎏金光晕，用于特殊按钮
```

## 组件样式类

### 按钮

```html
<!-- 主按钮 (墨绿渐变 / 夜间鎏金渐变) -->
<button class="btn-primary">主操作</button>

<!-- 次级按钮 (鎏金) -->
<button class="btn-secondary">次要操作</button>
```

### 卡片

```html
<!-- 普通卡片 -->
<div class="card-soft">...</div>

<!-- 焦点卡片 (带鎏金顶边) -->
<div class="card-focus">...</div>
```

## 使用示例

### 导航按钮

```tsx
// 激活状态: 墨绿 (日间) / 鎏金 (夜间)
<button className="text-primary dark:text-secondary">
  首页
</button>
```

### VIP 标识

```tsx
// 鎏金等级标识
<span className="bg-secondary text-white">S</span>
```

### 警告信息

```tsx
// 警示红
<div className="text-destructive border-l-2 border-destructive">
  关系风险预警
</div>
```

## 颜色语义

- **墨绿**: 沉稳、专业、信任
- **鎏金**: 尊贵、价值、重要
- **宣纸白**: 雅致、清爽、呼吸感
- **静夜蓝**: 深邃、专注、高端

---

**设计灵感来源**: 中国传统文化中的"儒商"理念，墨绿象征诚信与稳重，鎏金代表财富与尊贵。
