针对“新中式”风格的 7 点优化策略：
清晰文本对齐：
❌ 错误：所有文字居中（像写对联，难阅读）。
✅ 修正：左对齐为主。数字与标题严格左对齐，建立阅读秩序。
权重区分：
✅ 策略：字号倍率。核心数字（资产）用 32px，标签用 12px，形成 2.5倍反差。颜色的深浅（浓墨 vs 淡墨）区分层级。
主次操作：
✅ 策略：实心 vs 空心。底部的麦克风用“实心朱砂红”，其他的“查看全部”用“空心/文字按钮”。
圆角嵌套：
✅ 公式：外圆角 = 内圆角 + Padding。
如果卡片圆角是 20px，Padding 是 16px，那么里面的按钮圆角应该是 4px 或 8px，视觉才和谐。
柔和阴影 (关键)：
❌ 错误：在米色背景上用黑色阴影（显脏）。
✅ 修正：暖色阴影。使用 rgba(60, 40, 30, 0.08)，模拟物体投射在宣纸上的感觉。
统一间距：
✅ 策略：8pt 网格系统。所有间距都是 4 的倍数（8, 12, 16, 24, 32）。
内容突出：
✅ 策略：容器法。不要只改文字颜色，给重要信息加一个浅色背景容器（如红色背景的 10% 透明度）。
