## ✅ 1. 清晰文本对齐

**原则**: 左对齐为主，建立阅读秩序

### 已应用位置
- ✓ **Header**: 日期、问候语严格左对齐
- ✓ **全天焦点**: 标题和操作按钮左右对齐
- ✓ **关系预警**: 列表内容左对齐
- ✓ **常规事务**: 标题左对齐

### 代码示例
```tsx
<div className="flex flex-col items-start">
  <h1 className="text-[28px] font-serif">早安，李董事长</h1>
</div>
```

## ✅ 2. 权重区分

**原则**: 字号倍率 2.5倍，浓墨 vs 淡墨

### 字号层级
| 元素 | 字号 | 用途 |
|------|------|------|
| 主标题 | 28px | 页面标题"早安，李董事长" |
| 卡片标题 | 18px | 人名、重要标题 |
| 节标题 | 16-18px | "全天焦点"、"关系预警" |
| 正文 | 12-14px | 描述信息 |
| 辅助文字 | 10-12px | 次要信息、标签 |

### 颜色层级
- **浓墨**: `text-foreground` (主要内容)
- **中墨**: `text-primary/secondary` (强调内容)
- **淡墨**: `text-muted-foreground` (辅助信息)

## ✅ 3. 主次操作

**原则**: 实心 vs 空心，主次分明

### 按钮层级
1. **主操作 (实心)**
   - FAB 全能录入：墨绿渐变/鎏金渐变
   - 签到记录：实心主色按钮
   
2. **次操作 (空心/文字)**
   - 查看档案：空心边框按钮
   - 查看全部：纯文字按钮
   - 激活：空心红色按钮

### 代码示例
```tsx
{/* 主操作 - 实心 */}
<button className="bg-primary text-primary-foreground">签到记录</button>

{/* 次操作 - 空心 */}
<button className="bg-transparent border border-border">查看档案</button>

{/* 次操作 - 文字 */}
<button className="text-secondary font-bold">查看全部 ›</button>
```

## ✅ 4. 圆角嵌套

**原则**: 外圆角 = 内圆角 + Padding

### 嵌套规则
| 外圆角 | Padding | 内圆角 | 实际应用 |
|--------|---------|--------|----------|
| 20px | 20px (p-5) | 8px | VIP 焦点卡片 |
| 16px | 12px (p-3) | 8px | 警告卡片 |
| 16px | 16px (p-4) | 8px | 常规卡片 |

### 代码示例
```tsx
{/* 外圆角 20px，padding 20px */}
<div className="rounded-2xl p-5">
  {/* 内按钮圆角 8px */}
  <button className="rounded-lg">...</button>
</div>
```

## ✅ 5. 柔和阴影

**原则**: 暖色阴影，模拟宣纸投影

### 阴影系统
```css
/* 日间模式 - 暖色阴影 */
--shadow-soft: 0 2px 8px -2px rgba(60, 40, 30, 0.06), 
               0 8px 24px -6px rgba(60, 40, 30, 0.08);

--shadow-elevated: 0 4px 12px -2px rgba(60, 40, 30, 0.08), 
                   0 16px 40px -8px rgba(60, 40, 30, 0.12);

--shadow-float: 0 10px 30px -10px rgba(60, 40, 30, 0.15);

/* 鎏金光晕 */
--shadow-gold: 0 4px 14px 0 rgba(212, 175, 55, 0.3);
```

### 使用场景
- **柔和阴影**: 普通卡片 (`.card-soft`)
- **提升阴影**: 悬浮卡片、FAB
- **漂浮阴影**: VIP 焦点卡片 (`.card-focus`)
- **鎏金光晕**: 等级徽章、FAB 按钮

## ✅ 6. 统一间距

**原则**: 8pt 网格系统 (4 的倍数)

### 间距标准
| Tailwind Class | 实际值 | 用途 |
|----------------|--------|------|
| `gap-2` / `space-y-2` | 8px | 最小间距 |
| `gap-3` / `space-y-3` | 12px | 列表项间距 |
| `gap-4` / `space-y-4` | 16px | 卡片内容间距 |
| `gap-6` / `space-y-6` | 24px | 节之间间距（旧） |
| `gap-8` / `space-y-8` | 32px | 节之间间距（新） |
| `py-6` | 24px | 垂直内边距 |

### 已应用
- ✓ VIP 卡片横向间距: `gap-4` (16px)
- ✓ 卡片内容间距: `mb-4` (16px)
- ✓ 列表项间距: `space-y-3` (12px)
- ✓ 节间距: `space-y-8` (32px)

## ✅ 7. 内容突出

**原则**: 容器法，加浅色背景突出重点

### 应用实例

#### 1. 破冰策略容器
```tsx
<div className="bg-muted/50 p-3 rounded-lg border-l-2 border-primary">
  <span className="text-primary font-bold">💡 破冰策略：</span>
  {task.title}
</div>
```

#### 2. 等级徽章容器
```tsx
<span className="bg-destructive/10 text-destructive px-1.5 py-0.5 rounded">
  S级
</span>
```

#### 3. 日期标签容器
```tsx
<div className="border border-primary/30 px-2 py-0.5 rounded-sm">
  12月9日 · 宜拜访
</div>
```

#### 4. 鎏金等级徽章
```tsx
<div className="bg-secondary rounded-full shadow-gold">
  S
</div>
```
